# AgriWarung v2.5.23 - QR, iPhone Print, Help & Drawer Fix

Fokus patch:
- Mengembalikan struk dan label dari barcode ke QR code karena QR lebih stabil untuk kamera HP.
- QR berlaku untuk struk, batch inventori, produksi, panen, kegiatan kebun, dan hasil peternakan.
- Print iPhone/iPad memakai halaman print khusus 80mm agar tidak mencetak seluruh halaman Kasir.
- Thermal Bluetooth tetap tersedia untuk Android/Desktop; iPhone otomatis fallback ke print browser 80mm.
- Drawer dirapikan ulang dan menu default dibuat lebih ringkas.
- Tutorial & Bantuan diisi panduan bawaan: Kasir, Warung, QR, Print iPhone, Inventori Batch, Varian, Keuangan, Bon, Pajak, Role/Keamanan, Troubleshooting.

Catatan teknis:
- Tidak ada perubahan backend wajib pada patch ini.
- Frontend saja cukup untuk GitHub/Vercel.
- Backend-only zip tetap disediakan sebagai arsip konsistensi, tetapi HF tidak wajib diupdate jika sudah v2.5.22.
